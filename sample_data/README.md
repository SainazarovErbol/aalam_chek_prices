**60 товаров**, цены в сумах (примерные).

## Как загрузить в Google Таблицу

1. Открой [sheets.google.com](https://sheets.google.com) → **Создать** → пустая таблица
2. Переименуй первый лист в **«Молочная продукция»**
3. Открой файл `01_Молочная_продукция.csv` в блокноте → **Ctrl+A, Ctrl+C**
4. В Google Таблице выдели ячейку **A1** → **Ctrl+V**
5. Добавь новый лист (**+** внизу) → назови **«Бакалея»**
6. Вставь данные из `02_Бакалея.csv` — и так для всех 6 файлов

**Или через импорт:**
- Файл → Импорт → Загрузить → выбери CSV → «Заменить текущий лист» / «Создать новый лист»

## После загрузки

1. Поделись таблицей с email сервисного аккаунта Google (роль **Читатель**)
2. Скопируй ID таблицы из URL в `SPREADSHEET_ID` в `.env` / Railway
3. В боте: `/reload` — должно показать **60 позиций** по 6 отделам
